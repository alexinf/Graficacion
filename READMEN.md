Dillinger is a cloud-enabled, mobile-ready, offline-storage, AngularJS powered HTML5 Markdown editor.

  - Javascript
  - See HTML in the right
### Use
 - open the index.html file directly
 - select the color and press double click to color the worksheet    
 - press the draw button and mark two points, then press the button with the algorithm you want to use. and the canvas will be drawn.
 - to draw the circuit enter the radio and press the CIRCLE BRES button
 - TO CLEAN EVERYTHING PRESS THE BUTTON CLEAN SCREEN
### Development

Want to contribute? Great!

Dillinger uses Gulp + Webpack for fast developing.
Make a change in your file and instantanously see your updates!
Open your favorite Terminal and run these commands.
